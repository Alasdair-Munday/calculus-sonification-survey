What is sonification?

Sonification is the use of sound to represent information. This project uses sounds to represent different features of mathematical graphs. The sounds are designed to help the listener see how the vertical (Y) values relate to the horizontal (X) values.

There are three different features represented in the sound:

  • Y value
  • Slope (how Y is changing as X changes
  • Curve (how slope is changing as X changes)

What it sounds like

Choose the different graphs below to see how the different values affect the sound. Use the playback buttons or hover your pointer over the graph.

The Y value is matched to the pitch of the note. This means that if you are listening to a higher point, the note will be higher. Musically speaking: The note changes by two octaves from the bottom of the graph to the top, with the middle octave at 0.
